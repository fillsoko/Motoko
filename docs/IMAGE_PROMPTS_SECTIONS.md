## Style Notes for All Images

- **Strictly black and white** — no color, no sepia, no tinting
- **Editorial / documentary feel** — not stock photography, not staged
- **Dark and moody** — these will be used as background images behind white text, so they need to be predominantly dark (60-70% dark tones)
- **No faces clearly visible** — riders should be silhouettes or turned away
- **Frankfurt / Rhein-Main / Taunus region** aesthetic where applicable
- **Aspect ratio:** 16:9 (1920x1080)
- **File format:** Export as .webp, quality 80-85%
