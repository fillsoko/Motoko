# Section Background Images — motoko.de

Generate all images as **1920x1080px**, **black-and-white**, editorial motorcycle photography style. No text, no logos, no watermarks. Export as `.webp` and place in `src/assets/images/sections/`.

**Important:** These images are rendered at 10-15% opacity behind white text with a grayscale CSS filter. They need to be **predominantly dark** (60-70% dark tones) with enough tonal range to read as subtle texture, not a distraction.

**Base suffix for all prompts:** `black and white photograph, high contrast, cinematic film grain, Ilford HP5 pushed to 1600 ISO look, editorial motorcycle magazine style, 16:9 aspect ratio, no text, no logos, no watermarks, no visible faces`

## Style Notes for All Images

- **Strictly black and white** — no colour, no sepia, no tinting, no duotone
- **Editorial / documentary feel** — not stock photography, not staged or sterile
- **Predominantly dark** — 60-70% dark tones, these sit behind white text at low opacity
- **No faces clearly visible** — riders should be silhouettes, turned away, or cropped out
- **Frankfurt / Rhein-Main / Taunus region** aesthetic where applicable
- **Film grain** — think Ilford HP5 or Tri-X pushed, not clean digital
- **Aspect ratio:** 16:9 (1920x1080 minimum)
- **File format:** Export as `.webp`, quality 80-85%
- **Filenames:** `value-props-bg.webp`, `how-it-works-bg.webp`, `location-bg.webp`, `requirements-bg.webp`, `faq-bg.webp`, `instagram-bg.webp`, `cta-bg.webp`
