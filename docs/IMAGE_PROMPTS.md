## Tips

- **Midjourney**: append `--ar 21:9` or `--ar 16:9` and `--style raw` for editorial look
- **DALL-E / ChatGPT**: specify dimensions in the prompt and request "photographic, not illustrated"
- **Stable Diffusion**: use a realistic photography model (e.g., Juggernaut XL), negative prompt: "cartoon, illustration, text, watermark, logo"
- The site applies CSS `grayscale` filter automatically, so color images also work
- Place generated files in the paths listed above, then the site will pick them up automatically
